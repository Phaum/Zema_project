# Цифровая платформа экспресс-оценки коммерческой недвижимости

## Описание проекта  

Данный проект представляет собой веб-платформу для рыночной экспресс-оценки коммерческой недвижимости (бизнес-центры) на территории Санкт-Петербурга.  

Система реализует полный цикл оценки объекта недвижимости: от ввода исходных данных пользователем до получения итоговой рыночной стоимости.

Пользователь взаимодействует с платформой через веб-интерфейс, где он может:

- ввести характеристики объекта оценки
- указать кадастровый номер здания и земельного участка
- получить автоматически загруженные данные из государственных и открытых источников
- запустить расчет

На основе введённых и полученных данных система:

- подбирает рыночные аналоги
- анализирует их характеристики и местоположение
- рассчитывает арендную ставку
- определяет доходность объекта
- вычисляет рыночную стоимость

Для повышения точности оценки применяется:

- корректировка цен
- анализ ближайшего окружения
- учет инфраструктуры (метро, транспорт)
- статистические методы подбора аналогов

Результат предоставляется пользователю в виде:

- точечной оценки стоимости
- (при необходимости) диапазона значений

Дополнительно система позволяет визуализировать рыночные данные и использовать аналитические инструменты.

Проект представляет собой прототип аналитической платформы, объединяющей методы обработки данных, геоинформационного анализа и экономической оценки недвижимости.

## Инструкция по установке

1. **Клонирование репозитория**
```
git clone https://github.com/Phaum/Zema_project.git  
cd Zema_project
```

2. **Установка зависимостей**

**Frontend**
```
cd client  
npm i  
npm start
```
**Backend**
```
cd ../server  
npm i  
npm start 
```

## Переменные окружения

Для запуска необходимо создать файлы `.env` в папках `client` и `server`

**client/.env**

```
REACT_APP_API_PREFIX=http://localhost:8080/api 
```

**server/.env**

```
PORT=8080 
CLIENT_URI=http://localhost:3000 
PYTHON_BIN=python 
NSPD_PARSER_PATH=./scripts/nspdparser.py 
GEO_SERVICE_URL=http://127.0.0.1:8000/calculate 
GEO_HEALTH_URL=http://127.0.0.1:8000/health 

JWT_SECRET=your_jwt_secret 
DB_HOST=localhost 
DB_PORT=5432 
DB_NAME=zema_db 
DB_USER=your_db_user 
DB_PASSWORD=your_db_password 
DB_LOGGING=true
```

## Структура проекта
```
client/ 				# клиентская часть (React) 
	|--components/ 		# переиспользуемые компоненты 
    |--pages/ 			# страницы приложения 
    |--hooks/ 			# пользовательские хуки 
    |--context/ 		# глобальное состояние 
    |--utils/			# вспомогательные функции 
    
server/ 				# серверная часть (Node.js) 
	|--controllers/ 	# обработка запросов 
	|--services/ 		# бизнес-логика и расчёты 
	|--models/ 			# модели базы данных 
	|--routes/ 			# API маршруты 
```

- В `client` реализован пользовательский интерфейс и взаимодействие с системой (страницы, формы ввода данных, отображение результатов)
- В `server` реализована основная логика приложения, включая расчёты и обработку данных (API, расчёты, работа с данными и алгоритмы оценки)
- Ключевая логика расчёта стоимости находится в `services` 

## Информация о разработке

Проект разработан по техническому заданию в рамках реализации стартап-проекта, направленного на создание веб-платформы экспресс-оценки коммерческой недвижимости.  
В процессе разработки была спроектирована и реализована архитектура клиент–серверного приложения, включающая пользовательский интерфейс и серверную часть с бизнес-логикой.  

Особое внимание уделено реализации следующих компонентов системы:

- разработке ETL-процессов (сбор, парсинг и нормализация данных), включая:
	- определение ближайшего окружения объекта
	- расчет расстояния до ближайшей станции метро
	- определение ближайшей станции метро
	- определение территориальной зоны
	- определение принадлежности объекта к историческому центру
- разработке алгоритмов подбора аналогов
- организации взаимодействия между клиентской и серверной частями
